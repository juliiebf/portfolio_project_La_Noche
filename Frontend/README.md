# La Noche - Frontend

## Description
La Noche est un site web pour un bar karaoké situé à Paris dans les caves voûtées de Pigalle. Ce frontend propose une interface intuitive, immersive et responsive permettant de naviguer entre l'accueil, la réservation, les boissons, les chansons, la galerie, les événements, et la page de contact. Il intègre une fenêtre modale pour la connexion et l'inscription des utilisateurs.

---

## Technologies utilisées

- **HTML5** : pour la structure sémantique et l’accessibilité du contenu.
- **CSS3** : gestion du design, du responsive, des animations légères et des interactions visuelles.
- **JavaScript natif** : pour les interactions utilisateur simples, notamment l’ouverture/fermeture de modales et la gestion d’onglets.
- **Git** : contrôle de version du code source.
- (Optionnel selon projet) **React.js** ou autre framework si utilisé dans d’autres parties du projet.

---

## Description du HTML

- Le document utilise une structure claire avec un `<header>` contenant un logo cliquable et une barre de navigation principale.
- Navigation principale avec liens vers différentes pages du site telles que "Accueil", "Réservation", "Boissons", "Chansons", "Galerie", "Événements" et "Contact".
- Bouton "Se connecter" qui ouvre une fenêtre modale de connexion/inscription.
- Section principale (`<main>`) avec une zone héro contenant logo, slogan, et un bouton d’appel à l’action pour réserver.
- `<footer>` simple et épuré avec informations de contact et lien vers la page contact.
- Modale popup gérant les formulaires de connexion et de création de compte, avec onglets pour basculer entre les deux modes.
- Scripts JavaScript réduits directement intégrés pour gérer l’ouverture/fermeture de la modale et le changement d’onglets.

---

## Description du CSS

- Thème sombre avec fond noir omniprésent pour une ambiance de bar karaoké.
- Palette de couleurs dominée par :
  - Jaune/beige doré (#eddca2) pour les titres et éléments accentués.
  - Bleu vif (#22aaff) pour les boutons, liens survolés, et éléments interactifs.
  - Textes en gris clair pour un bon contraste sur fond sombre.
- Mise en page flexible utilisant Flexbox pour header, navigation, body et footer.
- Typographie soignée avec polices modernes `'Segoe UI', 'Montserrat', Arial, sans-serif`.
- Effets visuels sur les liens de navigation (soulignement animé, changement de couleur).
- Boutons stylisés avec arrondis, ombres portées, et effets au survol pour une UX réactive.
- Section héro avec fond noir, bord arrondi, et titres en jaune/beige lumineux.
- Tableau vidéo, cartes cocktails avec design harmonieux et transitions douces.
- Formulaires intégrés dans une modal avec design sombre, champs lisibles et boutons visibles.
- Responsive design pour s’adapter aux écrans à partir de 900px et en dessous jusque vers 650px, avec modification des grilles et ajustement des espacements.

## Installation & Démarrage

1. Cloner le dépôt :
git clone https://github.com/juliiebf/portfolio_project_La_Noche.git


2. Se placer dans le dossier :
cd portfolio_project_La_Noche


3. Si des dépendances (npm) sont utilisées, installer :
npm install


4. Lancer le serveur de développement (si applicable) :
npm start
